APRESENTAÇÃO





Este sistema foi desenvolvido pra realizar Consultas de Bens e Valores no Sistema de Patrimônio da Patribens.

O sistema foi desenvolvido em PHP e JavaScript e traz os dados de um banco SQL. Para esta apresentação esta página foi desenvolvida no HTML básico e em JavaScript. Os dados nesta página contêm descrição e valores fictícios e são carregados diretamente. Por isso foi colocada a mensagem de que os dados foram carregados. Outra mensagem que é mostrada é a conclusão da busca de mostrar todos os anos porque como o sistema traz as informações de um banco de dados este tempo de resposta não é instantâneo. Outro aspecto aqui é que as imagens (logomarca e setas) foram substituídas por simulações. Não existe atualização destes dados por aqui. Esta apresentação mostra apenas uma conta, 55/4800 - VEICULOS DIVERSOS, que contém veículos motorizados e não motorizados.

De início são apresentados o cabeçalho e as contas do sistema para a consulta.
