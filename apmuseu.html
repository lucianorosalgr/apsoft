APRESENTAÇÃO





Este sistema foi desenvolvido pra automatizar a venda de ingressos para o Museu.

O sistema foi desenvolvido em PHP e JavaScript e utiliza um banco SQL. Para esta apresentação esta página foi desenvolvida no HTML básico e em JavaScript. Os dados nesta página contêm dados e valores fictícios e são carregados diretamente. Por isso foi colocada a mensagem de que os dados foram carregados. É um sistema de venda de ingressos num museu que administra tanto pessoal para as funções do sistema quanto para venda de ingressos, apresenta as vendas na tela. Se precisar mostrar ao cliente mostra na tela exclusivamente a respectiva venda. Gera ainda um relatório das vendas de um determinado período que pode ser de um dia ou mais e também define os valores dos ingressos. Inicialmente foi formada uma massa de dados para geração desse relatório.

De início são apresentados o cabeçalho uma tela de entrada no sistema com matrícula, senha e botão de entrada.

Inicialmente o sistema já deve ter um usuário administrador cadastrado com poder de atuar em todas as funções do sistema, tanto de pessoal quanto relatórios e vendas. Os usuários podem ter funções de operadores, que apenas realizam as vendas, quanto de administrador que define os preços dos ingressos, geram relatórios, administram os usuários do sistema e até mesmo podem atuar também com venda de ingressos.

Quando o usuário for utilizar o sistema pela primeira vez deve colocar sua matrícula e sua senha que devem ser atualizados para poder entrar no sistema. A senha é igual a matrícula. Portanto é importante que o primeiro uso seja feito imediatamente seguido ao cadastro do usuário no sistema. Ou o cadastro seja feito pouco tempo antes do uso. Deve ser utilizada a matrícula, a senha atual e assim dois campos para a nova senha, sendo mais uma para confirmação. E por fim um botão para atualização da senha.

Para evitar algum desentendimento nos ingressos com quem comprou os ingressos pode ser mostrada a compra na tela de forma que seja apresentada ao cliente. Nesta tela anterior onde são mostradas os ingressos vendidos, basta clicar na compra que a esta tela do dia será apagada e mostrada apenas a compra feita. E ali a venda dos ingressos pode ser cancelada. Para cancelar a venda pode ser clicado no botão de cancelar. Depois disto, para continuarem as vendas basta clicar nesta parte da tela que apresentou a venda que volta a tela das vendas do dia.

Ao encerrarem as vendas por aquele usuário, seja ao final ou pela troca de usuário, ou mesmo para observação de um administrador, clicar no botão "Fim" e voltará a tela de entrada no sistema.

Quando o administrador entra no sistema outras operações podem ser feitas. Uma delas é o relatório de vendas que mostra as vendas realizadas num período a ser definido. Será apresentada a solicitação das datas de início e fim.
