APRESENTAÇÃO





Este sistema é desenvolvido pra realizar Cadastros e Consultas sobre os Estagiários da IPAES.

O sistema está sendo desenvolvido em PHP e JavaScript e utiliza um banco de dados SQL. Para esta apresentação esta página foi desenvolvida no HTML básico e em JavaScript. Os dados contidos nesta página são fictícios como nomes, endereços, números de documentos, contatos etc. e são carregados diretamente. Por isso foi colocada a mensagem de que os dados foram carregados. Nesta apresentação será considerada apenas as partes dos cadastros. Outro aspecto aqui é que as imagens (logomarca, lupa e chave) foram substituídas por simulações.

Esta apresentação será feita apenas nos cadastros de dados, dos estagiários, das instituições de onde vêm e dos setores desta instituição onde realizam suas atividades.

Vemos assim o seguinte menu:

Cadastros


Para iniciar o processo cada estagiário deve preencher um formulário que serviu de modelo a esta página, atendendo assim os requisitos de identificação e participação do estagiário nesta instituição.

O menu de cadastro foi desenvolvido com base neste formulário.

Assim é apresentada a opção relacionada ao estagiário na qual se referem às telas relativas aos campos do formulário:








Ao se solicitar uma tela relativa ao estagiário inicialmente será mostrada uma tela para a identificação pelo CPF. Pode-se ver que as mesmas opções do menu relacionados ao estagiário também aparecem abaixo nesta tela, facilitando assim o preenchimento ou consulta dos dados, não sendo necessário acessar através do menu suspenso.

Ao se preencher o campo do CPF será aberta a tela para entrada/consulta/alteração dos dados. Com a digitação do número completo pode-se digitar a tecla [enter], a tecla [tab] ou se clicar no botão da chave e a tela chamada anteriormente será apresentada.

Outro aspecto nesta tela é a a descrição da tela, neste caso informando que se trata do estagiário e no final ao lado direito um botão com um "X" para se fechar a tela com um clique. Outras formas de fechar a tela é digitando a tecla [escape] com o cursor no campo do CPF e o botão de cancelar ao final.

Os botões de Lupa mostram uma área na tela com as respectivas listas de acordo com o campo precedente. São telas com listas dos estagiários, das instituições e das diretorias.

Clicando na lupa ao lado do CPF aparecerá na tela uma lista dos estagiários em ordem alfabética, inicialmente com os que estão em atividade. Ao passar o mouse sobre o nome deles haverá um destaque. Cada nome terá uma cor de fundo indicando se o estagiário é de nível médio, superior ou técnico. Cada nível é indicado por duas cores para destacar as linhas quando estão consecutivas. Sendo em tons de vermelho para nível superior, de amarelo para nível médio e tons de verde para nível técnico. São apenas 20 nomes por tela, se o nome procurado não estiver apresentado, abaixo existem botões indicando passar pra frente ou pra trás a lista. Ao se clicar em um nome os dados do mesmo serão apresentados nas telas solicitadas.
