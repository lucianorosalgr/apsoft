APRESENTAÇÃO





Este programa foi desenvolvido pra automatizar a consulta do CATSER. A consulta apresenta além dos dados o valor do código do CATSER que é muito importante para que ser usado nas licitações de contratação de serviços. Utiliza apenas o HTML e JavaScript. Os dados nesta página estão disponíveis na internet, no endereço https://www.gov.br/compras/pt-br/acesso-a-informacao/consulta-detalhada/planilha-catmat-catser/catser-out2021.xlsx/@@download/file/Catser%20-%20Dez2021.xlsx em uma planilha para download. O volume de dados tem pouco mais de 2600 registros, foram todos colocados diretamente no programa em um array por serem apenas de texto, o que significa para a capacidade da informática atual um pequeno volume de dados. Ao se fazer uma pesquisa o resultado rapidamente é apresentado. A busca é feita com filtros do que se deseja e o que não se deseja buscar, os quais são informados ao programa pelos campos dos termos na tela.

Basicamente esta é a tela de se fazer a consulta. Para se fazer uma busca basta preencher os termos requeridos na tela do que se deseja que conste e que não conste nos registros que descrevem os materiais.

Por exemplo, se deseja buscar um monitor lcd com conexão em vga apenas se preenche o campo Com os Termos: com o valor software php. O resultado será o seguinte:


1/2
Grupo:1111 - SERVIÇOS DE DESENVOLVIMENTO DE SOFTWARE
Código:- 25860 - DESENVOLVIMENTO DE NOVO SOFTWARE - PHP
 Ativo (Linha na Planilha Original - 8)
2/2
Grupo:1112 - SERVIÇOS DE MANUTENÇÃO EVOLUTIVA DE SOFTWARE
Código:- 25933 - MANUTENCAO EVOLUTIVA DE SOFTWARE (ACRESCIMO DE NOVAS FUNCIONALIDADES) - PHP
 Ativo (Linha na Planilha Original - 15)


Se por acaso algum termo não interessar na busca ou se deseje que não apareça na consulta precisa apenas se preencher o campo Sem os Termos:, por exemplo, seja desejado o termo pintura e não se desejem os termos veiculo automotivo metal polimento, basta preencher e se refazer a busca. O resultado será:


1/5
Grupo:-7 - INVALIDO
Código:- 13455 - OBRAS CIVIS - PEQUENAS OBRAS / PINTURA EM GERAL
 Ativo (Linha na Planilha Original - 220)

Código:- 1090 - PINTURA INDUSTRIAL
 Ativo (Linha na Planilha Original - 331)
3/5
Grupo:5473 - SERVIÇOS DE PINTURA
Código:- 19372 - PRESTACAO DE SERVICOS DE PINTOR
 Ativo (Linha na Planilha Original - 332)
4/5
Grupo:5473 - SERVIÇOS DE PINTURA

Este é um exemplo simples. Mas a depender da busca podem aparecer milhares de resultados, assim o uso destes termos de filtragem podem se tornar extremamente necessários.
